# Malenia Grub Theme

## Installation

Clone the repository:
```
$ git clone https://github.com/jon4hz/malenia_grub_theme
```
Switch to the repository folder:
```
$ cd malenia_grub_theme
```
Run the installation script:
```
$ ./install.sh
```
